Salesforce Setup

How do I connect my Salesforce account?

  1. Log in to your Salesforce instance
  2. Go to Setup (top right)
  3. Search for "Connected Apps"
  4. Create a new app with OAuth enabled
  5. Get your Client ID and Client Secret
  6. Paste them into Calebrate's settings
Note: Your Salesforce credentials are never shared or stored outside your GitHub repository. They're encrypted and used only for API calls to validate prospect ownership.

GitHub Setup

How do I connect my GitHub account?

Required Permissions
  1. Personal Access Token (read/write to your private repository)
  2. Access to "github.com/CalebGarritson/Prospect-tracker" repo
Creating a PAT
  1. Go to GitHub Settings (click your profile icon, top right)
  2. Click "Developer settings" → "Personal access tokens"
  3. Click "Generate new token (classic)"
  4. Give it a name like "Calebrate"
  5. Select scopes: repo, user:email, read:org
  6. Generate and copy the token
  7. Paste it into Calebrate
Never share your PAT. It's your authorization key to GitHub. If you accidentally expose it, regenerate it immediately on GitHub.
Tip: Your PAT is stored in your browser's local storage and never sent to external servers (except GitHub API calls).
