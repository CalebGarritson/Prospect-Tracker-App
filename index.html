🔐 Connect to GitHub

Your prospect data lives in your private GitHub repository. Enter your GitHub username and a Personal Access Token (PAT) to get started. Your data is never lost — even if you clear your browser cache.

Need a PAT? Go to github.com/settings/tokens/new → set expiration → check repo scope → click Generate.
