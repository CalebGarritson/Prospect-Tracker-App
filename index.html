Welcome to Calebrate

A prospect tracker for Gusto sales reps. All data stays in your private GitHub repo.

Finding Your Salesforce User ID

Your Salesforce User ID is a unique identifier in your Salesforce profile.

Steps:

1. Log in to Salesforce

2. Click your profile icon (top right)

3. Select "Settings"

4. Copy the User ID from your profile (starts with "005")

5. Paste it above

Why?

Calebrate validates that you own each lead in Salesforce before creating a task. This prevents accidental task creation on leads you don't own.

GitHub Setup Instructions

Calebrate stores all your data in a private GitHub repository. No servers, no accounts.

Step 1: Create a Personal Access Token

1. Go to github.com/settings/tokens

2. Click "Generate new token (classic)"

3. Give it a name like "Calebrate"

4. Check these scopes:

repo (full control of private repositories)

user (read user profile data)

5. Click "Generate token"

6. Copy the token (you won't see it again)

Step 2: Enter Your Credentials

1. Enter your GitHub username (e.g., "CalebGarritson")

2. Paste the token you just generated

3. Calebrate will create a Prospect-tracker repo in your account

4. All prospect data will be stored there, encrypted in your control
