How to Find Your Salesforce ID ✕

In Salesforce:

  1. Click on your profile photo or avatar in the top-right corner
  2. Select Your Name to open your profile
  3. Look for your User ID starting with 005
  4. Copy the full 15–18 character ID and paste it above

Why we need this:

Your Salesforce ID confirms you own the leads in your Prospect Tracker. Calebrate blocks tasks for leads owned by other reps — keeping everyone's data secure.

Connect Your GitHub Account

Step 1: Create a GitHub Personal Access Token

  1. Go to github.com/settings/tokens
  2. Click Generate new token (classic)
  3. Give it a name like "Calebrate"
  4. Select scopes: repo (full control of private repositories)
  5. Click Generate token and copy it immediately — you won't see it again

Step 2: Create Your Data Repository

  1. Go to github.com/new
  2. Name it Prospect-tracker (must be exact)
  3. Make it Private
  4. Click Create repository
  5. In your new repo, create a data folder with these files:
    • data/prospects.json with []
    • data/focus.json with []
    • data/settings.json with {}

Step 3: Link Calebrate

  1. Return here and enter:
    • GitHub Username: your GitHub username
    • Personal Access Token: the token you copied in Step 1
  2. Complete setup with your profile info

All set!

Calebrate will now sync all your prospect and focus data directly to your private GitHub repository. No data leaves your control.
